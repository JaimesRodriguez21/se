### Instalacion 

Requisitos
----------

- Nodejs 14.0 o superior'

#### Clonar repositorio
`https://github.com/JaimesRodriguez21/send-email.git`

#### Instalar dependencias

`$ npm install ` 


#### Configurar variable de entorno

`$ .env `  Ejemplo en `$ copy.env ` 

#### Ejecutar

`$ node app.js `
